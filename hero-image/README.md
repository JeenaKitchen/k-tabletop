# Hero Image Directory

This directory contains the custom hero image for the landing page.

## How to Use:

1. **Add your custom image** to this folder
2. **Name it**: `your-custom-image.png` (or update the filename in LandingPage.js)
3. **Recommended size**: 400x400 pixels (square aspect ratio)
4. **Supported formats**: PNG, JPG, WebP

## Current Path in Code:
```jsx
<img src="/hero-image/your-custom-image.png" alt="Your Custom Hero Image" />
```

## To Change the Filename:
Update the src path in `/src/components/LandingPage.js` to match your actual filename.

Example:
```jsx
<img src="/hero-image/my-hero-image.jpg" alt="My Hero Image" />
```
